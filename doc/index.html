Luminosity Masks

  • What are luminosity masks?

    Luminosity masks are a way to control images based on its tones. Confused? See this excellent article by Tony Kuyper. This script is based on his great work.

  • How to install this script?

    • If you want to try this script just once open Photoshop and go to File->Scripts->Browse...
      Choose the file Marc Sàbat Luminosity Masks.jsx.

      (Restart Photoshop if needed)

    • If you want to install it permanently and be easy accessible just copy the file Marc Sàbat Luminosity Masks.jsx to the scripts folder.
      Usually: c:\Program Files\Adobe\Adobe Photoshop CC (64 Bit)\Presets\Scripts\

  • How does it work?

    Upon firing the script a dialog will be presented:

    Here you can choose how many depth levels of luminosity masks will be created and wheter or not curves adjustment layers will be created based on luminosity masks channels.

    Once the script has finished you will find the new luminosity channels:

    You can load any channel as a selection (using the corresponding button at the bottom of channels tab) and, then, create any adjustment layer. The new adjustment layer will have attached the selected luminosity mask.

    If you decided to create curves adjustment layers you will find three different sets (Darks, Lights and Midtones). Each set will contain as many adjustment layers as the depth specified.

  • Why did you create this script?

    Multiple reasons:

    • I did not find a free script or action easily downloadable (by that I mean no registration or giving away eMail address).
    • Some actions I found had issues with different UI languages (i.e. did not work for Photoshop installations different than the language they were created in).
    • I wanted some control like how many luminosity masks are going to be created and be able to create curves adjustment layers automatically.
